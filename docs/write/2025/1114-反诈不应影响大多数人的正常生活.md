# 反诈不应影响大多数人的正常生活

![](./src/20251114.png)

一个制定政策或者执行政策时应该明白的道理：过犹不及！

当下的中国，虽然移动支付的普及率高，但是纸币仍然没有完全退出市场，取现是再正常不过的事。用户把钱存到银行或者使用银行的转款业务，理应可以自由取现。更重要的是，银行不会也没能力跟踪用户取现后的实际使用情况，从这个层面来说银行柜台的反诈能力是很弱的，你可以按政策要求做一些询问、备注，但做多了就是刁难用户。

柜台工作人员询问用户银行卡流水，那就更是不应该了。银行应该有反诈、反洗钱的风控管理，对于有风险的账户做出限制甚至报警处置，无论如何也轮不到银行的柜台工作人员来询问吧？既然用户的账户是正常的，那么调取用户的银行卡流水，要求用户就某一笔进账的来源做解释，这难道不是刁难吗？

对于银行而言，尽量减少自己的责任边界，那不是更应该寻求的吗？
