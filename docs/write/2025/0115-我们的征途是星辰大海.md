# 我们的征途是星辰大海

>北京时间2025年10月31日23时44分，搭载神舟二十一号载人飞船的长征二号F遥二十一运载火箭在酒泉卫星发射中心点火发射，约10分钟后，飞船与火箭成功分离，进入预定轨道。
>北京时间2025年11月1日3时22分，成功对接于空间站天和核心舱前向端口，整个对接过程历时约3.5小时，创造了神舟飞船与空间站交会对接的最快纪录。
>北京时间2025年11月1日4时58分，神舟二十一号航天员乘组入驻中国空间站。


从仰望星空、文学（影视）作品的无穷想象，到载人航天飞行、空间站上科研人员长期在轨驻留，人类切切实实的迈出了一大步。

文学、影视作品中的虫洞、空间跳跃技术等依然只是想象，遥远的空间距离限制着人类，火星移民在未来数十年乃至数百年或许都难以实现。

神舟二十一号载人飞船从发射到航天员入驻空间站，用时不到5个半小时，进步神速，空间站商业化，大量接待地球游客，感觉未来几年就可能实现吧，有钱人旅游有了新选择。

中国对月球的探索大概才是未来对外太空探索的模板：中国探月工程初期规划为绕、落、回三期。绕：2004～2007年（一期）研制和发射首颗月球探测卫星，实施绕月探测。落：2013年前后（二期）进行首次月球软着陆和自动巡视勘测。回：2020年前（三期）进行首次月球样品自动取样返回探测。后期启动探月工程四期，实现月球背面软着陆及采样返回，规划构建月球科研站基本型，开展月球环境探测等任务。

在月球上建成有人入驻的科研站以及后续商业化接待更多人，大概就是当下的科技能保证人类正常迁移的最远距离了吧。至于更远星球的探索，基本都要以无人为主了。未来无人驾驶的飞船在太空中穿越，探索星球、建立根据地，等待地球人的到来。

中国的航天事业起步虽晚，却取得了伟大的成就，未来探索外太空，中国应该是主力，在遇到更加强大的外星文明之前，说汉语是宇宙通用语也不为过。

至于外星移民，那应该还要很久，即便技术上实现了，人类适应地球之外的环境也需要很久很久吧。但也相信未来的某一天，身处其他星球的人，终其一生也回不到遥远的地球。
