# 今天，明天，昨天

>为什么会发生这种事？……如果我这么做而不是那么做呢？……是我的错……为什么这些事会发生在我身上？……其他人都有美好的生活……我被抛下了……现在改变已经太晚了……事情不会有转机了……我过去很幸福，但现在已经快乐不起来了，永远都不会了……我什么时候失去了快乐的能力？……我希望事情是另一番景象…… 
>——萨拉·埃德尔曼《胡思乱想消除指南：用认知行为策略走出情绪困境》


> 我以前一直不懂，为什么有人会把看过的电影再看一遍，都知道后续剧情了，还有什么意思。我无所事事地随便翻看订阅时，找到了这部七年前看过的电影，讲的是主角为了和喜欢的女生在一起，一次次穿越时空的故事，这部好莱坞电影当年评价很高，重看一遍之后，我明白了一件事，正因为知道后续剧情，才能以旁观者的视角俯瞰，想着这里要是那样做就好了。这时我忽然意识到，说起来，我也一直在为这种事耿耿于怀。
> ——《灰原同学重返过去，开启所向无敌的第二轮青春游戏》

日复一日的找不到出路，不停的在原地转圈圈

当下的人生过的不如意，对明天小小的期待就像一排排气球。当一个又一个明天成为今天，就好似在不停的戳破前面的气球，让那小小的期待破碎。

当下的人生过的不如意，当一个有一个今天成为昨天，过去就是过不去的坎。要是...如果...时不时的出现在脑海中。
