# 今天，明天，昨天

>为什么会发生这种事？……如果我这么做而不是那么做呢？……是我的错……为什么这些事会发生在我身上？……其他人都有美好的生活……我被抛下了……现在改变已经太晚了……事情不会有转机了……我过去很幸福，但现在已经快乐不起来了，永远都不会了……我什么时候失去了快乐的能力？……我希望事情是另一番景象……




>明日复明日，明日何其多。
>我生待明日，万事成蹉跎 。
>世人若被明日累，春去秋来老将至。
>朝看水东流，暮看日西坠。
>百年明日能几何？请君听我明日歌。


> 我以前一直不懂，为什么有人会把看过的电影再看一遍，都知道后续剧情了，还有什么意思。我无所事事地随便翻看订阅时，找到了这部七年前看过的电影，讲的是主角为了和喜欢的女生在一起，一次次穿越时空的故事，这部好莱坞电影当年评价很高，重看一遍之后，我明白了一件事，正因为知道后续剧情，才能以旁观者的视角俯瞰，想着这里要死那样做就好了。这时我忽然意识到，说起来，我也一直在为这种事耿耿于怀。——《灰原同学重返过去，开启所向无敌的第二轮青春游戏》
